Tomcat7 Realm Setup Documentation available online at 

http://tomcat.apache.org/tomcat-7.0-doc/realm-howto.html

The basic steps are

1.  Install MySQL
2.  Start MySQL and create  database (with name 'authority' by default)
3.  Create user and user_role tables.  Can use db/mysql55/userroletables.sql to create these tables.
4.  Add user and role information to the user and user role tables.
5.  Add mysql connector jar (in tomcat7/lib/mysql-connector-java-5.1.22-bin.jar) to <tomcathome>/lib.
6.  Change the Realm setup in <tomcathome>/conf/server.xml.  See at the bottom of
the file tomcat7/conf/server.xml for example changes.
7.  Restart Tomcat7

See http://tomcat.apache.org/tomcat-7.0-doc/realm-howto.html for more details.
